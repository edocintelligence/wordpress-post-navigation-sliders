EADd Advanced Post Navigation Flyouts for next and previous posts with hover, featured image thumbnail support, and background color selection.

== Description ==

Engage users with Post Navigation Fly-ins for Next and Previous posts in specific categories. Select the categories for the plugin to use, and Next and Previous side slide-in animations will draw readers deeper.

= Easy Table Setup =

Select the categories you want to have the post navigation sliders working in, select the background color, and you are set. Simple and fast way to increase reader depth on your website.

== Installation ==

1. Upload the `edoc-post-navigation-sliders` folder to the `/wp-content/plugins/` directory
1. Activate the eDoc Post Navigation Sliders plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `eDoc Post Sliders` menu that appears in your admin menu

== Frequently Asked Questions ==

* Can I use this in multiple categories?
	* Yes! Just select the correct categories for display and it will display on the sides for them.
* Can I prevent this from displaying in certain post categories?
	* YES! Hold control + click to select ONLY the category to display this in on Windows. On mac command + click. Unselected categories will not display the post navigation sliders.
